# RECON-Z 🦅

RECON-Z is an all-in-one OSINT & Recon toolkit built in Python by Zeshan Haider (@TurboHacks110). It combines domain intel, subdomain discovery, social media footprinting, image metadata extraction, and phone number intelligence — all in one terminal tool.

📂 GitHub Repo: https://github.com/turbohacks/Recon-Z

==============================
🔥 FEATURES
==============================
✅ WHOIS Lookup (domain details, registrar info)  
✅ IP Address Lookup (location, ISP, Google Maps)  
✅ Username OSINT (Facebook, IG, YouTube, X)  
✅ Subdomain Finder (auto wordlist – no file needed)  
✅ Image Metadata Extractor (EXIF, resolution, GPS)  
✅ Phone Number Intelligence (region, carrier, timezone)  
✅ All results can be saved to .txt files  
✅ One-color terminal UI: Bright Green Theme
